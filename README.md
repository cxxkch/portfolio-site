# 🎬 Studio Portfolio

Сайт-портфолио с админ-панелью.

---

## 📋 Пошаговая инструкция по запуску

### Шаг 1 — Установи Node.js

Скачай и установи Node.js с официального сайта:  
👉 https://nodejs.org (выбери версию LTS)

Проверь установку — открой терминал (или cmd) и введи:
```
node -v
npm -v
```
Если показало версии — всё ок.

---

### Шаг 2 — Скачай проект

Скачай папку `portfolio-site` к себе на компьютер.

---

### Шаг 3 — Установи зависимости

Открой терминал, перейди в папку проекта и запусти:
```bash
cd portfolio-site
npm install
```
Подожди пока установятся все пакеты.

---

### Шаг 4 — Запусти локально (проверка)

```bash
npm run dev
```
Откроется ссылка вида `http://localhost:5173` — открой её в браузере.  
Если сайт работает — переходи к деплою!

---

### Шаг 5 — Создай GitHub-репозиторий

1. Зайди на https://github.com и войди в аккаунт (или создай новый)
2. Нажми **"New repository"** (зелёная кнопка)
3. Назови его `portfolio-site`
4. Оставь **Public**, не ставь галочки на README/gitignore
5. Нажми **Create repository**

---

### Шаг 6 — Залей проект на GitHub

Установи Git, если ещё нет: https://git-scm.com

В терминале в папке проекта:
```bash
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/portfolio-site.git
git push -u origin main
```
Замени `ТВОЙ_ЮЗЕРНЕЙМ` на свой никнейм на GitHub.

---

### Шаг 7 — Деплой на Vercel (бесплатно)

1. Зайди на https://vercel.com
2. Нажми **"Sign Up"** → войди через GitHub
3. Нажми **"Add New..." → "Project"**
4. Найди свой репозиторий `portfolio-site` и нажми **Import**
5. Настройки оставь по умолчанию (Vercel сам определит Vite)
6. Нажми **Deploy**
7. Через ~1 минуту сайт будет доступен по адресу вида:  
   `https://portfolio-site-xxx.vercel.app`

---

### Шаг 8 (опционально) — Свой домен

В Vercel → Settings → Domains → добавь свой домен.  
Vercel покажет DNS-записи, которые нужно прописать у регистратора домена.

---

## 🔐 Админ-панель

- Кнопка **admin** находится в самом низу сайта (еле видна)
- Пароль по умолчанию: `admin123`
- Чтобы сменить пароль, измени значение `ADMIN_PASSWORD` в файле `src/App.jsx`

---

## ⚠️ Важно

Сейчас данные хранятся в `localStorage` браузера. Это значит:
- Данные сохраняются только у того, кто зашёл
- Отзывы посетителей видны только им самим
- Админ-изменения видны только в твоём браузере

Если нужна полноценная база данных (чтобы все видели одни данные), нужно будет подключить бэкенд (например Firebase, Supabase или свой сервер).
